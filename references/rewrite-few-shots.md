# Rewrite Few-shots

只学改写动作，不借原句。

## 适用范围

仅用于起草模式和改稿模式。

## 判定单位

- 句级：用于识别高风险句式（L0）
- 段级：用于识别模板密度、信息密度（L1/L2）
- 文级：用于 Gate 通过判定

## 高风险句式（L0 候选）

以下句式命中后，先记为 L0 候选：

- `不是 A，而是 B`
- `不在 A，也不在 B，而在 C`
- `虽然 A，但 B`
- 同段连续两句使用同一转折骨架

## 误伤豁免

命中 L0 候选后，满足任一条件可降为 L1：

- 句子后半部分给出新增具体信息（场景/动作/数据）
- 该句为用户明确指定保留原句
- 该句用于引用他人原话且不可改写

未满足豁免条件，判定为 L0。

## 分级规则

### L0（拦截级）

命中任意一条：

- 未豁免的高风险句式
- 连续两句以上模板化转折且无新增信息
- 段尾使用空结论收束（无新增判断）

处理要求：必须重写，清零前不得输出终稿。

### L1（强修级）

命中任意一条：

- 转折可读但机械，段内重复同义判断
- 段内判断多、支撑少（缺场景/动作/例子/数据）
- 句子信息真实但表达偏“口号化”

处理要求：优先改关键句，尽量降到 L2 或清零。

### L2（优化级）

命中任意一条：

- 节奏单一，句长变化不足
- 用词抽象，可替换为更具体表达
- 小标题信息量偏低

处理要求：时间允许时优化，不阻塞输出。

## 计数口径

- L0：按“句”计数
- L1：按“段”计数（每段最多记 1 条）
- L2：按“段”计数（每段最多记 1 条）

## 门禁清单（输出前必检）

### Gate 1：硬门禁

- L0 条目数必须为 0
- 不得出现连续模板化转折链
- 结尾必须落到明确判断、提醒或行动建议

任一项不通过：继续改写，不输出终稿。

### Gate 2：质量门禁

- L1 条目数 <= 2（按段计）
- 每个一级段至少一处具体信息（场景/动作/例子/数据任一）
- 开头两段内必须出现主题判断

任一项不通过：继续改写。

### Gate 3：发布门禁

- 语气前后统一
- 无明显模板腔（整齐排比、空话堆叠、机械转折）
- 标题与正文主判断一致

任一项不通过：做最后一轮清扫。

## Few-shot

反例：

`问题不在错别字，也不在观点跑偏，而在整篇文章虽然都说对了，却不像我写的。`

改写：

`真正让我警觉的，是这篇文章没有明显错误，却还是不像我写的。`

反例：

`这不是文笔问题，而是流程问题。`

改写：

`我后来发现，真正卡住我的，是写作流程。`

反例：

`信息没错，语气也算顺，但发出去后很快就收到一句反馈：像模板拼出来的。`

改写：

`信息没出错，行文也算顺。可文章一发出去，我很快就收到一句反馈：像模板拼出来的。`

反例：

`逻辑没问题，结构也完整，但整篇文章读完没留下印象。`

改写：

`这篇文章的逻辑和结构都过关。问题是，读完以后几乎没留下印象。`
