# Lessons Learned

用于记录每次公众号写作任务的短复盘。

开始任务前先快速浏览最近几条。
结束任务后追加，不覆盖历史。

## Entry Template

### YYYY-MM-DD | 主题

- Goal: 这篇文章想达成什么
- Worked: 这次最有效的结构、表达或策略
- Avoid: 这次暴露出的低效写法或不合适方向
- Reuse Next Time: 下次可以默认沿用什么

## Entries

### 2026-05-17 | 技能初始化

- Goal: 搭建一个纯 Markdown 驱动的公众号长文写作技能
- Worked: 把流程拆成讨论、提纲、起草、改稿、复盘五种模式，并把长期偏好与单次经验分开存放
- Avoid: 用额外脚本、配置或复杂记忆机制把技能做重
- Reuse Next Time: 先读取 `author-profile.md` 和 `lessons-learned.md`，完成任务后再做轻量追加
